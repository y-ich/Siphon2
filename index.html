Editor for Programming

Siphon2 is powered by CodeMirror, which has many syntax highlight modes for programmers.Lines are foldable according to languages' syntax.

Siphon2 is a web app, it means that it is not necessary to install an app and references on Web are in next tabs.

Input Enhancement 1

Emmet(ex-Zen Coding) for HTML is available.

Input Enhancement 2

Object-based auto completion for JavaScript and CoffeeScript.

REPL on Editor

Read-Eval-Print-Loop on the editor for both JavaScript and CoffeeScript.

Cloud Access

Sign-in and authenticate for Dropbox. Your cloud files will be on Siphon2.

You can choose permission between "App folder" limited and full access of your Dropbox folder.

NOTE: Currently only test users can sign-in for Dropbox because the app is in beta test phase. If you are interested in using Dropbox features, please contact us.

CoffeeScript/LESS compile

You can enable an option to upload a CoffeeScript or LESS file with its compiled JavaScript or CSS file.

And More

  • Auto save and resume of buffers
  • Responsive layout for landscape
  • Full screen web app capable for more wide screen
  • powered by CodeMirror, Dropbox, Emmet, CoffeeScript, LESS, jQuery, Underscsore.js, and Bootstrap
  • Local file import on desktop WebKit browsers
